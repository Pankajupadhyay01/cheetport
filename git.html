Copy code For Being Efficent


  1. To check git version 
                        
git -version
  2. To add mail in new git account 
                        
 git config -global user.mail "mail id here "
  3. To change git detail 
                        
 git config -global -edit
  4. To upload file/folder into github 
                        
git init
  5. To add file into staging area 
                        
git add <name of file without bracket>
  6. To commit file in github 
                        
git commit -m "message here"
  7. To check commited files 
                        
git log
  8. To check change inside the repo 
                        
git status
  9. To discard the commit 
                        
git checkout <hash code of that commit eithout bracket>
  10. To create branch 
                        
git branch <name of branch you want to make without bracket >
  11. To check branches 
                        
git branch
  12. To move to main or another branch 
                        
git checkout /*name of branch or hash code */
  13. To merge two branch 
                        
git merge /*name of branch you want to merge*/
